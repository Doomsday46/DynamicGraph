Для запуска генерации и добавление случайных записей в базу данных, необходимо нажать на кнопку "Генерация записей в бд". 
Для отображение графика необходимо пройти по ссылке "Перейти к графику", за тем нажать на кнопку отобразить. В такой последовательности действий, 
записи буду с некоторой периодичностью добавляться в базу данных и добавление новых данных будет отображатся на графике(график обновляется с заданной, программно, периодичностью).
